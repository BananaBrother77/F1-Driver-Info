# F1 Driver Info Dashboard

A modern, responsive web dashboard that displays real-time Formula 1 driver information using the **OpenF1 API**. This project showcases driver profiles, team names, and dynamic team branding.

## Features

* **Real-time Data:** Fetches latest driver sessions from the OpenF1 API.
* **Dynamic Branding:** UI elements (driver numbers, borders, and hex codes) automatically update based on the official team colors.
* **Smart Sorting:** Drivers are automatically sorted by their permanent racing number.
* **Mobile Optimized:** Fully responsive grid layout that adapts to any screen size.
* **Performance:** Built with vanilla JavaScript using `async/await` and optimized API filtering to prevent duplicates.

## Tech Stack

* **HTML5:** Semantic structure and SEO metadata.
* **CSS3:** Custom properties (variables), Grid Layout, and Flexbox.
* **JavaScript:** Fetch API and dynamic DOM manipulation.
* **API:** [OpenF1](https://openf1.org/) (Live Formula 1 timing data).
